

🧠 BienestarMind - Gestión de Bienestar al Aprendiz
BienestarMind es un sistema de información diseñado para el programa de formación ADSO del SENA. Su propósito es centralizar y automatizar los procesos de bienestar, facilitando la interacción entre aprendices, instructores y bienestar al aprendiz.

📌 Descripción del Proyecto
La plataforma permite gestionar de manera eficiente:

Asesorías: Registro y seguimiento de citas de bienestar.

Fichas y Programas: Control de la información académica vinculada a los usuarios.

Recursos: Reserva de espacios físicos y elementos (equipos, herramientas) dentro del centro.

Roles: Diferenciación de permisos para Administradores, Instructores y Aprendices.

🛠️ Tecnologías y Requisitos
Para el desarrollo y despliegue local se requiere:

Lenguaje: Python 3.14 o superior.

Framework: Django 5.x.

Base de Datos: PostgreSQL.
